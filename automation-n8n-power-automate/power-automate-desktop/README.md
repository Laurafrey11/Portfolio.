# Power Automate Desktop – Web Data Download Pipeline

## Overview

This project implements an automated data pipeline using **Power Automate Desktop** to download data from a web source and store it securely in **SharePoint**.

The automation eliminates manual interaction by navigating the website, downloading the required data files, and uploading them directly to a designated SharePoint document library.

---

## Project Description

The Power Automate Desktop flow performs the following steps:

- Accesses a web-based data source
- Navigates through the required pages
- Downloads the target data files
- Stores the files in a SharePoint location
- Ensures consistency and repeatability of the process

This solution is designed to improve efficiency, reduce human error, and support scheduled or recurring executions.
